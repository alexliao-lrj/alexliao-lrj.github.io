使用Eclipse Oxygen + maven3 + tomcat7搭建Springboot开发环境

Prepare

  • 开发环境：win7 + Eclipse Oxygen + maven + tomcat7
  1. Install Java, configure system environment variable. Use cmd java -version to check your version.
  2. Install tomcat. Visit http://localhost:8080 to check if server started successfully.
  3. 配置maven
    • 下载并解压maven
    • 修改系统环境变量，Path最后加上%maven_home（maven安装路径）%\bin;
    • 修改\conf\settings.xml，设置maven repository路径，以后的依赖都下载在这里:
      <localRepository>E:\maven_repo</localRepository>
    • cmd mvn -v可以查看maven版本和安装信息，顺便检验是否安装成功
  4. 配置Eclipse + maven
    • Help - install new software下载m2e插件，装完重启eclipse生效
    • Window - preference - Maven - user settings使用自己的\conf\settings.xml，顺带更新Local Repository路径
  5. 配置Eclipse + Tomcat
    • Window - preference - Server - Runtime Environment Add自己的Tomcat
