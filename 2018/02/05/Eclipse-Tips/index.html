Eclipse Tips

发表于 2018-02-05
Trivial problems I met about Eclipse.

  1. 更改jre版本：右键project->build path->configure build path->libraries->选中JRE System Library，选择右侧Edit->alternate jre->选择自己安装的jdk(java 1.8)

  2. 创建的maven项目没有src/main/resources文件夹(application.yml所在文件夹)：右键project->build path->configure build path->Source->选中src/main/java，add folder，新建一个resources文件夹

  3. Eclipse YAML插件：YEdit，官网：https://github.com/oyse/yedit，离线版本：（链接: https://pan.baidu.com/s/1b1j2gQ 密码: wyyb）安装方法：直接复制JAR包到Plugins文件夹，重启eclipse即可。

  4. cannot find mappers in package”a0”, 是mybatis配置没能让系统扫描a0(主程序App)之外的包，不知道怎么改，先在app上手动加mapperscan，强行指定mappers的位置

  5. Description:

Cannot determine embedded database driver class for database type NONE

没有连接数据库，application.yml里看datasource mysql部分格式有没有写对

  1. postman：post请求raw方式，后台无法识别
