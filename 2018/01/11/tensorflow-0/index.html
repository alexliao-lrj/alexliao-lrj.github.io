在win7下安装tensorflow

1. 安装Anaconda

我们通过Anaconda来安装，首先安装Anaconda，从TUNA下载：
https://mirrors.tuna.tsinghua.edu.cn/help/anaconda/

tensorflow在python3.5环境下对windows的适配较好，因此选择anaconda3-4.2，自带python3.5，而从4.3版本开始内置默认版本python3.6，win下可能会踩很多坑。
安装时勾选自动配置环境变量，免去修改系统环境变量的麻烦。

安装完成，测试：

  • 测试python：

    -cmd python 正确进入shell

  • 测试Anaconda：

    -cmd conda info 显示相关配置信息

安装完后可以使用TUNA的Anaconda仓库镜像，以后安装其它包速度会快一些(避免wall)，cmd执行2个命令：

conda config --add channels 'https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/free/'

conda config --set show_channel_urls yes

2. 安装tensorflow

tensorflow有CPU和GPU两种版本，GPU安装复杂一些，需要装一个叫cuda的东西。我的电脑现在还没有独显，就装cpu版本的，也比较简单：

管理员身份打开cmd，输入

pip install --upgrade --ignore-installed tensorflow

测试：

输出正确结果 2

可能会报错：
C:\tf_jenkins\home\workspace\rel-win\M\windows\PY\35\tensorflow\core\platform\cpu_feature_guard.cc:137] Your CPU supports instructions that this TensorFlow binary was not compiled to use: AVX

问题原因是你的CPU是支持的，但tensorflow没有按照支持AVX、AVX2、FMA等编译。不解决demo也可以正常运行，网上的解决方案：

  • 官网下载tensorflow源码
  • 用bazel编译
  • 用pip安装.whl文件包，如果已经安装了anaconda，想把编译生成的.whl文件安装在anaconda下面的tensorflow环境中：
    • source activate TensorFlow #这个是在anaconda环境下创建的环境名字
    • pip install 用bazel编译生成的.whl
    • source deactivate

3. Pycharm配置

如果以前已经装过python，现在想要在pycharm里直接使用tensorflow，进行如下修改：

打开pycharm file->settings->project interpretor 修改成anaconda目录下的python.exe
